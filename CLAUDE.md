# Website Creation

## Tech Stack

- Next.js 15 + TypeScript + Tailwind CSS
- Framer Motion for animations

## Design Rules

- Use the AskUserQuestion tool to interview the user about the website design so that you can accurately reflect the user's ideas
- Use the frontend-design skill for all UI decisions
- Use UI/UX Pro Max for design system generation
- Use 21st.dev for component inspiration if needed/provided
- No generic AI aesthetics
- Bold, distinctive design choices
- Performance-optimized (Core Web Vitals)
